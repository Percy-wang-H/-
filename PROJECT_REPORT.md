# 华商创新班成绩管理系统项目说明

## 项目背景

本系统用于管理创新实验班课程成绩，解决 Excel 成绩表分散、统计不直观、查询不方便的问题。

## 系统功能

1. 成绩数据导入：支持上传课程平台导出的 Excel 文件。
2. 成绩总览：展示学生人数、平均分、最高分、及格率。
3. 学生管理：支持新增、编辑、删除、搜索和排序学生成绩。
4. 图表分析：展示综合成绩分布和各项成绩平均值。
5. 数据导出：支持将当前成绩表导出为 CSV。
6. 作业提交：生成 GitHub 链接、公网链接和系统功能说明文字。

## 数据来源

数据来自课程平台导出的 `23本计算机科学与技术（创新实验班）【1-2】班_统计一键导出.xlsx`。系统内置 56 名学生的综合成绩示例数据，并可通过导入功能替换为新的成绩表。

## 技术实现

- 页面结构使用 HTML 编写。
- 样式使用 CSS 实现响应式后台界面。
- 交互逻辑使用 JavaScript 实现。
- Excel 文件解析使用 SheetJS。
- 图表使用 Canvas 绘制。

## 部署方式

本项目是静态 Web 项目，可以部署到 GitHub Pages、阿里云 ECS + Nginx 或任意静态网站托管服务。
